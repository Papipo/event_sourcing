# EventSourcing

EventSourcing, as its name suggests, is an Actor based Event Sourcing library for ruby.

## Installation

Add this line to your application's Gemfile:

    gem 'event_sourcing'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install event_sourcing
